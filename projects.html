Projects

  • CS21x is a work in progress project. I am developing an assignment autograder and a gradebook using CS50's open source tools for MIT's Introduction to CS course.
  • MiniBloq is a visual programming interface which I enhanced to program Electric Ray Robot, a low cost robotic platform. A curriculum has also been designed for anyone starting with programming. More about the project can be read on this research publication.
  • Flappy is a game engine build using C++ and OpenGL and has support for scene graph, model loading, textures, lights, cameras and shaders. A naive implementation of the Flappy Bird game using the game engine can be found here.
  • Cake O' Clock is a simple website I developed for my friend to help with her cake shop. All artworks were designed using Inkscape.
  • Resource Allocation Graph Simulator is a tool I developed to teach Deadlock concepts when I was the TA for Operating Systems course. You can see a demo of the tool here.
